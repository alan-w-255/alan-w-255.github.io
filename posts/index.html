linux 单独为应用设置分辨率

通过设置 QT_SCREEN_SCALE_FACTORS 环境变量为每个视频输出端口设置分辨率。 如果我们需要单独为某一个应用设置这个变量，我们可以利用 .desktop 文件来实现启动某一个应用时单独设置分辨率。 找到 .desktop 文件的 Exec= 字段，在后面添加你要缩放的倍数。 例如： Exec=env QT_SCREEN_SCALE_FACTORS=2 firefox 然后保存文件，更新 .desktop 文件，输入命令 update-desktop-database。 这样，点击 firefox 图标启动时，就自动设置了 QT_SCREEN_SCALE_FACTORS=2 这个环境变量。应用程序分辨率放大了 2 倍。

Start Hugo Blog

安装 hugo sudo pacman -S hugo 创建新站点 hugo new site blog 安装主题 cd blog git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git theme/ananke echo 'theme = ananke' >> config.toml 添加第一篇博客 hugo new posts/my-first-post 查看效果 hugo server -D 然后打开浏览器，输入服务地址，就能看到效果啦。
